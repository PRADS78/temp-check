<Title>DisprzPortalDomProvider</Title>

<Description>
  This component provides way for Disprz Components to render the popup's
  outside the React root DOM. This resolves the z-indices issue for multiple
  components. Render this component to the root component of your application.
  All the popup's are added in div element with an id named `disprz-popper`
  under the body tag.
</Description>

<h3>Usage</h3>

```jsx
import { DisprzPortalDomProvider } from "@disprz/components";
function App() {
  return (
    <DisprzPortalDomProvider>
      {/* Your App goes here  */}
    </DisprzPortalDomProvider>
  );
}
```

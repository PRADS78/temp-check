<Description>
  As the type of button components increased we separated the buttons into
  separate components. Example:
</Description>

**Older version of v3 components**

We have a button with a prop **type** `outlined` here

```js
import { DisprzButton } from "@disprz/components";
function RenderButton() {
  return (
    <DisprzButton ctrCls="outlined-button" label="Outlined" type="outlined" />
  );
}
```

**Newer version of v3 components**

Here we have the same button as a separate component

```js
import { OutlinedButton } from "@disprz/components";
function RenderButton() {
  return (
    <OutlinedButton ctrCls="outlined-button" label="Outlined" />;
  );
}
```

All the components are named after their type

1. PrimaryButton
2. OutlinedButton
3. HyperlinkButton
4. PlainButton
5. DropdownButton
